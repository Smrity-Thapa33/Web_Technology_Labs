# 🌐 Web Technology Labs

This repository contains lab assignments for the **Web Technology** course. It includes practical implementations of frontend and backend concepts using HTML, CSS, JavaScript, PHP, XML, and database technologies.

---

## 📂 Lab Contents

---

### 🔹 Lab 1: HTML & CSS

- Create Home page with links (About Us, Services, Contact Us)
- Department description using paragraphs and lists
- Class timetable using tables
- Types of CSS (Inline, Internal, External)
- Different types of lists
- Web page layout using `<div>`
- Frame-based layout
- List box implementation
- Marksheet display
- Registration form
- Display image in webpage
- Create CV webpage

---

### 🔹 Lab 2: JavaScript & jQuery

- Alert message on button click
- Sum of two numbers using prompt
- Display current date & time
- Sum of even numbers (1–25)
- Form validation
- Arrays and foreach loop
- Email validation using regex
- Cookie operations (set, get, delete)
- Dropdown value using jQuery
- AJAX content loading
- jQuery chaining (slideUp/slideDown)
- Image slider
- Animation (width, height, opacity)

---

### 🔹 Lab 3: PHP & Database

- Handle divide-by-zero exception
- Operators (arithmetic, comparison, logical)
- Form handling and data display
- File reading and display in table
- Login system (authentication)
- Store student data in database
- Session-based page view counter
- SQL (create database, table, column)
- Cookies in PHP
- Classes and objects
- Passing variables via URL

---

### 🔹 Lab 4: XML & XSLT

- XML for Book information
- XML + XSD validation (Employee data)
- XML validation with constraints (book records)
- XML with XSLT (student list display)
- XML for songs data
- XML to HTML table using XSLT
- XML with CSS styling
- XML validation using DTD

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- jQuery  
- PHP  
- MySQL  
- XML, XSD, DTD, XSLT  
- VS Code  

---

## 🎯 Objectives

- Learn frontend and backend web development  
- Understand client-server interaction  
- Work with forms, validation, and authentication  
- Handle databases using PHP & SQL  
- Learn XML data representation and validation
